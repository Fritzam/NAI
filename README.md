Instrukcja wykonania zadania na zjazd 1: Knight's Duel.
W katalogu KnightsDuel znajduje się plik main.py, którego wywołanie uruchomi grę.
Przed wykonaniem pliku należy wywołać poniższe komendy:

Windows: python -m venv .venv
cd .venv/Scripts/Activate.ps1
pip install -r requirements.txt

Linux: python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
